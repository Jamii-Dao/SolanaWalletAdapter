# SolanaWalletAdapter
A lightweight Rust Solana Wallet that can be used in Rust based frontends and WebAssembly.

#### Features
- [x] Connecting to a wallet
- [x] Signing a message
- [?] Creating a Versioned Transaction
- [?] Sign In With Solana
- [?] Handling errors
- [ ] Send a transaction
- [ ] Parse a response

Those marked with `?` have partial support at the moment
